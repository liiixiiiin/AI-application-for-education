# 课程大纲：数据库基础（示例）

## 课程简介
本课程面向计算机专业基础阶段，帮助学生理解关系型数据库的核心概念与基本操作，能够完成简单的数据建模与 SQL 查询。

## 学习目标
- 理解关系模型与基本术语（表、行、列、主键、外键）
- 掌握 SQL 的基础查询与数据操作语句
- 了解规范化与常见数据异常

## 章节安排
### 第 1 章：关系模型与数据表
- 关系模型的基本概念
- 表结构设计与字段类型
- 主键与外键的作用

### 第 2 章：SQL 基础查询
- SELECT 与 WHERE 的基本用法
- 排序与分页
- 聚合函数与分组

### 第 3 章：数据完整性与规范化
- 完整性约束（实体、参照、用户自定义）
- 第一范式与第二范式
- 插入/更新/删除异常

## 教学建议
建议先通过实例数据集展示关系表，再引导学生完成基础 SQL 查询练习。
