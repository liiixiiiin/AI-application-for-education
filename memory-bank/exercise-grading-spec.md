# 练习评测规范（Step 7 输出）

## 1. 学生提交格式
```
{
  "exercise_id": "ex_001",
  "course_id": "course_001",
  "type": "single_choice",
  "answer": "B"
}
```

## 2. 评测输出格式
```
{
  "exercise_id": "ex_001",
  "correct": true,
  "score": 1,
  "feedback": "回答正确。",
  "suggestion": ""
}
```

## 3. 判定规则

### 3.1 客观题（规则匹配）
- **单选题**：答案与标准答案一致则正确。
- **判断题**：true/false 与标准答案一致则正确。
- **填空题**：答案与标准答案或 `alternatives` 列表中任一项匹配则正确（忽略大小写与首尾空白）。

### 3.2 主观题（语义相似度 + 大模型判分）

#### 3.2.1 语义相似度评分
使用 Embedding 模型将学生答案与参考答案编码为向量，计算余弦相似度：
```
sim(s, r) = (s · r) / (||s|| × ||r||)
```
分段阈值映射为分数：
- sim ≥ 满分阈值（0.85）→ 满分
- sim ≥ 及格阈值（0.60）→ 部分得分（线性插值）
- sim < 及格阈值 → 部分得分系数（0.6）× 线性比例

#### 3.2.2 大模型判分
将学生答案、参考答案与评分要点（rubric）送入大模型，输出归一化评分与评语。

#### 3.2.3 加权融合
最终得分 = α × 语义相似度分 + (1−α) × 大模型评分
权重系数 α 可根据题型特点调节（默认 0.4）。

#### 3.2.4 多要点拆分评分
对于含多个得分要点的简答题，将参考答案拆分为 n 个关键要点，分别计算匹配得分并加权求和：
```
total_score = Σ(w_i × score_i), i=1..n
```
其中 w_i 为各要点分值权重，score_i 为按上述方法计算的匹配得分。

## 4. 评测约束
- 输出必须包含 `correct`、`score`、`feedback` 字段。
- `score` 取值区间 0–1（便于统计正确率），部分得分可使用小数。
- 简答题需返回 `suggestion`（缺失点提示或改进建议）。

## 5. 辅助字段（可选）
```
{
  "matched_points": ["说明主键用于唯一标识记录"],
  "missing_points": ["说明外键建立表间引用关系"],
  "similarity_score": 0.78,
  "llm_score": 0.82,
  "final_score": 0.80
}
```
