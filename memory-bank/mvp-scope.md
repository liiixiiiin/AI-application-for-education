# 基础功能清单 + 不做项（Step 1 输出）

## 基础功能清单（MVP）
- 账号体系：注册 + 账号密码登录；三角色（管理员/教师/学生）均可访问最小功能页面
- 课程管理：教师创建课程；三角色可查看课程
- 知识库管理：课程关联知识库文档（支持一门课多文档）；支持 PDF/Word/Markdown 混合输入
- RAG 问答：基于课程知识库进行问答，答案包含知识点来源/片段引用
- 练习生成：单选/判断/简答题；附标准答案与知识点来源；简答题包含评分标准
- 练习评测：按标准答案判定正确/错误，并给出建议
- 基础统计：用户数、练习次数、正确率

## 不做项（MVP 不包含）
- 统计大屏与可视化看板
- 复杂学情分析（如知识点趋势、错题聚合、多维画像）
- 细粒度权限控制与审批流程
- 课件/资源批量导出
- 本地模型推理优化与多模型切换（先接入外部模型 API 完成闭环）
